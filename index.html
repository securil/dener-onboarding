목표: BNB(MOL 스왑에 사용)와 POL(수수료용 소액)을 마련합니다. 준비 규모 — 레퍼러 약 $650~800 / 총판 겸업 약 $960~1,100 상당.

경로 A — BNB를 직접 파는 거래소 (바이낸스, 빗썸 등)

  1. 거래소 가입 + 신원 인증(KYC)

  2. 원화(또는 USDT) 입금 후 BNB로 교환
    본드 규모 + 수수료 여유분 5~10%를 더해 마련하세요.
  3. POL도 $10~20 상당 마련
    .dex 도메인 등록과 Polygon 수수료(가스비)에 사용됩니다.

경로 B — BNB가 없는 거래소 (업비트 등)

  1. 전송 빠른 코인(XRP, TRX 등)으로 교환해 바이낸스로 전송
  2. 바이낸스에서 BNB로 환전

MetaMask 지갑 만들기 & 코인 입금

예상 20~30분BNB 체인Polygon

목표: 내 지갑(MetaMask)을 만들고, 두 개의 네트워크(BSC·Polygon)를 추가한 뒤, 거래소에서 코인을 옮겨 옵니다.

지갑 설치

  1. PC: 크롬 우측 상단 ⋮ → 확장 프로그램 → Chrome 웹스토어에서 MetaMask 검색 → 여우 아이콘 확인 후 Chrome에 추가
    모바일: 앱스토어에서 MetaMask 공식 앱 설치 (제작사 확인!)
  2. 새 지갑 생성 → 비밀번호 설정
  3. 시드 문구 12단어를 종이에 적어 보관
    이 12단어 = 내 전 재산의 열쇠. 사진 촬영·카톡 전송 금지, 누가 물어봐도 알려주지 마세요. 잃어버리면 복구 불가.

네트워크 추가 (길 2개 뚫기)

  1. MetaMask 상단 네트워크 선택 → 네트워크 추가 → 목록에서 BNB Smart Chain과 Polygon 추가
    목록에 없으면 chainlist.org 접속 → 검색 → Connect Wallet → Add로 자동 추가.

거래소에서 입금

  1. MetaMask 상단 계정명 클릭 → 내 주소 복사 0x…
  2. 거래소 출금 화면에서 BNB 출금 — 네트워크는 반드시 BSC (BEP-20) 선택
  3. POL 출금 — 네트워크 Polygon 선택
  4. 먼저 소액 테스트 출금 → 도착 확인 후 나머지 전액 출금
⚠️ 네트워크를 잘못 선택해 보내면 코인을 잃을 수 있습니다. 출금 직전 '네트워크' 항목을 두 번 확인하세요.

DEXignation에서 .dex 도메인 등록

예상 10~15분Polygon 체인

목표: 복잡한 지갑 주소 대신 쓸 나만의 인터넷 이름표(예: myname.dex)를 NFT로 소유합니다. 추천 링크와 총판 활동의 기반이 됩니다.

  1. dexignation.com/ko 접속 → 우측 상단 지갑 연결
    모바일은 MetaMask 앱 안의 '브라우저' 탭에서 접속하세요.
  2. 네트워크를 Polygon으로 전환하라는 안내가 나오면 승인
  3. 검색창에 원하는 이름 입력 → 사용 가능 여부 확인
    추천 링크에 쓰일 이름이니 짧고 기억하기 쉬운 것으로! (예: roy.dex)
  4. 등록 기간 선택 1·3·5·10·15년 → 결제 토큰 선택 POL / USDT / USDC
  5. 등록은 2단계입니다: ① 신청 서명 → ② 잠깐 대기(가로채기 방지 절차) → ③ 확정 서명
    중간에 창을 닫지 말고 안내에 따라 두 번째 서명까지 완료하세요.
  6. 완료 후 내 계정(내 도메인)에서 이름이 보이는지 확인 — 이 이름은 NFT로 내 지갑에 보관됩니다
만료 주의: 기간이 끝나기 전 갱신하면 계속 소유할 수 있습니다. 만료일을 캘린더에 등록해 두세요.

대표 도메인 등록 & 리워드 받을 BNB 지갑 연결

예상 10분Polygon 체인

목표: 방금 등록한 이름을 내 지갑의 '공식 이름표(대표 네임)'로 지정하고, 이름 뒤에 리워드가 정산될 BNB 지갑 주소를 적어 둡니다.

대표 도메인(대표 네임) 지정

  1. DEXignation 내 계정 → 보유 도메인에서 방금 등록한 이름 선택
  2. 대표 네임으로 지정 → MetaMask 서명
    지정하면 내 지갑 주소가 이 이름으로 표시되고, 추천 링크(ref=내이름.dex)에 사용됩니다.

BNB 수령 지갑 등록

  1. 내 도메인의 레코드 관리 열기
  2. 주소 레코드 추가 → 체인 목록에서 BNB (BSC) 선택
  3. 리워드를 받을 BNB 지갑 주소를 복사·붙여넣기로 입력 → 서명
    한 글자만 틀려도 리워드를 받지 못합니다. 직접 타이핑하지 마세요.
  4. digmol.com → 내 총판 대시보드에서 정상 표시 확인
왜 필요한가요? 모든 리워드는 BNB로 정산됩니다. 여기 등록한 주소가 실제 '리워드 수령 지갑'이 됩니다.

BNB → MOL 스왑, 생태계 참여 (Dener 서임)

예상 10분BNB 체인2,000만 MOL = 약 $620

목표: digmol.com에서 BNB를 MOL로 스왑해 생태계에 참여하고, 보유량 2,000만 도달로 Dener 서임을 받습니다. 서임은 블록체인에 기록되어 누구나 검증할 수 있습니다.

  1. 먼저, 나를 안내한 총판의 .dex 이름을 확인
    할인(최대 15%)을 받는 열쇠입니다. 모르면 스왑 전에 꼭 물어보세요.
  2. digmol.com 접속 → 지갑 연결 → 네트워크를 BSC로 전환
  3. 스왑(서임) 화면에서 할인 도메인 (.dex) 칸에 총판의 .dex 이름 입력
  4. 넣을 BNB 금액 입력 → 받을 MOL 수량 자동 계산 확인
    교환 비율은 컨트랙트에 새겨져 있어 흥정도 차별도 없습니다.
  5. MetaMask 서명 → MOL이 즉시 내 지갑으로
  6. 보유 MOL이 2,000만에 도달하면 서임 게이지가 100% — Dener 완료!
이후 일과: 추천 링크(app.digmol.com/ref?ref=내이름.dex)로 소개 활동 → 리워드는 매일 밤 10시 마감·11시부터 수령(Claim) 가능 → 안 찾아도 사라지지 않고 쌓입니다.
⚠️ MOL 교환 비율은 상장 전 고정 기준으로 운영됩니다. 정확한 현재 비율은 스왑 화면에 표시된 값을 기준으로 하세요.

자주 묻는 질문

리워드는 무엇으로 받나요?

전부 BNB로 받습니다. 광고를 보고 캐는 채굴 보상만 MOL이에요. 그래서 STEP 4에서 BNB 수령 지갑을 등록하는 것이 중요합니다.

리워드를 안 찾으면 사라지나요?

아니요. 블록체인에 계속 쌓이고 언제든 수령(Claim)할 수 있습니다. 매일 밤 10시 집계 마감 → 11시부터 수령 가능입니다.

지갑을 사이트에 연결하면 위험하지 않나요?

연결 자체는 '명함을 보여주는 것'과 같아 코인이 빠져나가지 않습니다. 코인이 움직이는 것은 '서명'을 눌렀을 때뿐이니, 서명 전에 내용(금액·사이트 주소)을 확인하는 습관만 들이면 됩니다.

총판 .dex 이름을 모르면 어떻게 하나요?

스왑 전에 나를 안내해 준 분(총판)에게 꼭 물어보세요. 입력하면 할인(최대 15%)을 받고, 안내한 총판에게 정당한 보상이 갑니다. 입력하지 않으면 할인도 없습니다.

내가 나를 총판으로 지정해도 되나요?

본인 지정은 금지되어 있습니다. 다른 참여자가 나를 지정해야 리워드가 발생합니다.

이 프로젝트는 믿을 수 있나요?

스마트컨트랙트는 외부 감사(Beosin) 2건을 완료했고, 코드가 GitHub에 전부 공개되어 있으며 PolygonScan에서 소스 검증이 완료되어 누구나 확인할 수 있습니다. 다만 MOL의 상장 전 고정 기준은 시장 가격이 아니므로, 무리하지 않는 범위에서 참여하시길 권합니다.

참여 준비 규모는 얼마나 되나요?

레퍼러 기준: 본드 $620 상당 BNB + .dex 등록비(사이트에서 확인, $10~50 예비) + 수수료 $5~10 = 약 $650~800 상당. 총판 겸업은 본드가 $930 상당이라 약 $960~1,100 상당입니다. (기준가 $0.000031, 스왑 화면 표시값이 우선)
