スワップに使うBNBを用意する

目安 30〜60分＋本人確認の待ち時間必要なもの：顔写真付き身分証明書 ・ 銀行口座

目標：BNB（MOLスワップに使います）と、少額のPOL（手数料用）を用意します。参加準備規模 — レファラーの場合はおよそ$750〜900相当、レファラー＋エバンジェリストの場合はおよそ$1,100〜1,300相当です。

ルートA — BNBを直接取り扱っている取引所（Binance、Bithumbなど）

  1. 口座を開設し、本人確認（KYC）を済ませる
    身分証明書の撮影と銀行口座の連携が必要です。確認には1〜2日かかることがあります。
  2. 入金したうえで、BNBに交換します
    コミット済みMOL（PV）の規模に加えて、手数料の余裕分として5〜10%多めに用意してください。
  3. POLも$10〜20相当だけ用意します
    .dexドメインの登録と、Polygonネットワークの手数料（ガス代）に使います。

ルートB — BNBを扱っていない取引所（Upbitなど）

  1. 送金の速い銘柄（XRP、TRXなど）に交換して、Binanceへ送金します
  2. BinanceでBNBに交換します
    POLはお使いの取引所で用意し、Polygonネットワークで出金することもできます。
⚠️ 国や地域によっては（たとえば韓国）、取引所からMetaMaskなどの個人ウォレットへ送金する際に、トラベルルールにもとづく送金先ウォレットの事前登録が必要です。必要になる前に、取引所の「個人ウォレット登録」メニューを確認しておいてください。
出金は今ではなく、STEP 2で行います。 まずウォレットを作り、アドレスを取得してから出金します。この順番を守ってください。

MetaMaskウォレットを作り、コインを移す

目安 20〜30分BNB ChainPolygon

目標：自分のウォレット（MetaMask）を作り、必要な2つのネットワーク（BSCとPolygon）を追加したうえで、取引所からコインを移します。

ウォレットをインストールする

  1. パソコン：Chromeのメニュー ⋮ → 拡張機能 → ChromeウェブストアでMetaMaskを検索 → キツネのアイコンであることを確認 → Chromeに追加
    スマートフォン：アプリストアから公式のMetaMaskアプリをインストールしてください（提供元の表示を必ず確認！）
  2. 新しいウォレットを作成 → パスワードを設定します
  3. 12個のシードフレーズを紙に書き、安全な場所に保管します
    この12語は、あなたの資産すべての鍵です。写真に撮らない、メッセージアプリで送らない、誰にも教えない。失くすと復元できません。

ネットワークを追加する（2本の道を通す）

  1. MetaMask上部のネットワーク選択をタップ → ネットワークを追加 → 一覧からBNB Smart ChainとPolygonを追加します
    一覧にない場合は、chainlist.org → 検索 → Connect Wallet → Add の順で自動的に追加できます。

取引所から出金する

  1. MetaMask上部のアカウント名をクリック → 自分のアドレスをコピーします 0x…
  2. 取引所の出金画面でBNBを出金します — ネットワークは必ずBSC (BEP-20)を選んでください
  3. POLを出金します — ネットワークはPolygonです
  4. まず少額でテスト送金 → 着金を確認 → そのうえで残りを送ります
⚠️ ネットワークを間違えると、コインが永久に失われることがあります。出金を確定する前に「ネットワーク」の欄を2回確認してください。

DEXignationで.dexドメインを登録する

目安 10〜15分Polygonチェーン

目標：長いウォレットアドレスの代わりに使える、読みやすいインターネット上の名前（たとえば myname.dex）をNFTとして持ちます。これが紹介リンクや、エバンジェリストとしての活動の土台になります。

  1. dexignation.com にアクセス → 右上のウォレットを接続
    スマートフォンでは、MetaMaskアプリ内の「ブラウザ」タブから開いてください。
  2. Polygonへの切り替えを求める案内が出たら承認します
  3. 検索欄に希望の名前を入力 → 使用できるかを確認します
    この名前は紹介リンクにも使われるので、短くて覚えやすいものにしましょう（例：roy.dex）。
  4. 登録期間を選びます 1・3・5・10・15年 → 支払いトークンを選びます POL / USDT / USDC
  5. 登録には2回の署名が必要です：① 申請の署名 → ② 少し待機（横取り防止のための手順） → ③ 確定の署名
    途中でウィンドウを閉じず、案内にしたがって2回目の署名まで進めてください。
  6. 完了したら、自分のアカウントに名前が表示されるか確認します — この名前はNFTとしてウォレットに保管されます
有効期限にご注意：期間が終わる前に更新すれば、そのまま持ち続けられます。有効期限をカレンダーに登録しておきましょう。

代表ネームを設定し、リワードを受け取るBNBウォレットを紐づける

目安 10分Polygonチェーン

目標：登録したばかりの名前を、自分のウォレットの正式な名札（代表ネーム）として指定し、その裏側にリワードが精算されるBNBアドレスを記録します。

代表ネームを設定する

  1. DEXignationで自分のアカウントを開き、保有ドメインから登録したばかりの名前を選びます
  2. 代表ネームに設定 → MetaMaskで署名します
    設定すると、ウォレットアドレスがこの名前で表示され、紹介リンク（ref=あなたの名前.dex）にも使われます。

BNB受取ウォレットを登録する

  1. 自分のドメインのレコード管理を開きます
  2. アドレスレコードを追加 → チェーンの一覧からBNB (BSC)を選びます
  3. リワードを受け取るBNBアドレスをコピー＆ペーストで入力 → 署名します
    1文字でも違うと、リワードは届きません。手入力はしないでください。
  4. digmol.com → エバンジェリストダッシュボードで、正しく表示されているか確認します
なぜ必要なのですか？ リワードはすべてBNBで精算されます。ここで登録したアドレスが、実際にリワードを受け取るウォレットになります。

BNBをMOLにスワップして、エコシステムに参加する（デナー叙任）

目安 10分BNB Chain2,000万MOL ≒ $720相当

目標：digmol.comでBNBをMOLにスワップしてエコシステムに参加し、MOLの保有量が2,000万に達することでデナーの叙任を受けます。叙任はブロックチェーンに記録され、誰でも検証できます。

  1. まず、あなたを案内してくれたエバンジェリストの.dexネームを確認します
    これが割引（最大15%）を受けるための鍵です。わからない場合は、スワップの前に必ず尋ねてください。
  2. digmol.com にアクセス → ウォレットを接続 → ネットワークをBSCに切り替えます
  3. スワップ（叙任）画面の割引ドメイン (.dex)欄に、エバンジェリストの.dexネームを入力します
  4. 入れるBNBの数量を入力 → 自動計算されたMOLの数量を確認します
    交換比率はコントラクトに書き込まれているため、交渉の余地も、人による差もありません。
  5. MetaMaskで署名 → MOLがすぐにウォレットへ届きます
  6. MOLの保有量が2,000万に達すると、叙任ゲージが100%になります — これでデナーです
そのあとの日々の流れ：紹介リンク（app.digmol.com/ref?ref=あなたの名前.dex）を共有 → リワードは毎日22時に集計が締まり、23時から受け取り（Claim）できます → 受け取っていないリワードは消えず、ブロックチェーン上に積み上がっていきます。
⚠️ MOLの交換比率は、上場前の固定された基準レートで運用されています。正確な現在の比率は、必ずスワップ画面に表示されている値をご確認ください。

エバンジェリストになる — 任意のステップ

3,000万MOL ≒ $1,080相当管理者による登録が必要

目標：参加者から指定を受けるたびに、その参加額の5%をリワードとして受け取るエバンジェリストになります。これはレファラーのリワードとは別枠で、同じ参加に対して両方が適用されます（両方の役割を持つ場合、1代の参加につき最大25%）。

  1. 保有量が3,000万になるまで、追加でMOLをスワップします（方法はSTEP 5と同じで、およそ$360相当の追加です）

    コミット済みMOL（PV）を保有しているだけでは、指定の対象にはなりません。管理者ページで有効化される必要があります。
  3. digmol.comでエバンジェリストダッシュボードを開き、参加者向けの割引率0〜15%を設定します（自分の5%は自動です）
  4. 参加者にお伝えします：「スワップ画面の割引欄に私の名前（あなたの名前.dex）を入れると割引になります」
    この一言が、エバンジェリスト活動のすべてです。指定は任意なので、お伝えしなければリワードは発生しません。
おすすめの進め方：まずレファラー（およそ$720相当）から始めて活動の流れをつかみ、そのあとエバンジェリスト（およそ$360相当の追加）へ広げる方法をおすすめします。月に5人から指定を受ける場合、その拡張分に相当するリワードがおよそ2か月で積み上がる計算になります（例であり、保証されるものではありません）。

よくある質問

リワードは何で受け取りますか？

すべてBNBで受け取ります。広告を見て採掘するマイニングのリワードだけがMOLです。だからこそ、STEP 4でBNBの受取ウォレットを登録することが大切なのです。

受け取らないでいると、リワードは消えてしまいますか？

いいえ。ブロックチェーン上に積み上がり続け、いつでも受け取れます。毎日22時に集計が締め切られ、23時から受け取りが可能になります。

ウォレットをサイトに接続するのは危なくありませんか？

接続そのものは名刺を見せるようなもので、コインは動きません。コインが動くのは「署名」を押したときだけです。ですから、署名の前に内容（数量やサイトのアドレス）を確認する習慣さえあれば大丈夫です。

エバンジェリストの.dexネームがわからない場合は？

スワップの前に、案内してくれた方に必ず尋ねてください。入力すると、あなたは割引（最大15%）を受けられ、相手には正当なリワードが渡ります。入力しなければ、割引はありません。

自分自身をエバンジェリストに指定できますか？

いいえ、自己指定は認められていません。ほかの参加者から指定を受けたときにのみ、リワードが発生します。

このプロジェクトは信頼できますか？

スマートコントラクトは外部監査（Beosin）を2件完了しており、コードはGitHubにすべて公開され、PolygonScanでソースの検証も済んでいるため、誰でも確認できます。ただし、上場前のMOLの固定基準レートは市場での相場ではありませんので、ご自身にとって無理のない範囲でご参加ください。

どれくらい準備すればよいですか？

レファラーの場合：コミット済みMOL（PV）のためのBNBが約$720相当＋.dexの登録費用（サイトでご確認ください。$10〜50を見込んでおくと安心です）＋ネットワーク手数料$5〜10 ＝ 合計およそ$750〜900相当です。エバンジェリストも兼ねる場合はコミット分が約$1,080相当になるため、およそ$1,100〜1,300相当となります。（基準レート$0.000036。スワップ画面に表示される値が優先されます。例であり、保証されるものではありません。）
